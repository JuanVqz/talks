## Marp Tips

Slides are built with [Marp](https://github.com/marp-team/marp-cli).

**Run slides locally (with live reload):**

```bash
PORT=3010 marp -p --server .
```

**Export to PDF:**

```bash
marp --pdf --allow-local-files slides.md
```

The `--allow-local-files` flag is required for local images to render correctly in the PDF.
